This *'Sparks'* app is intended to simplify different social sparks throughout the day.
Making life easier for any individual whom may suffer with a busy scheduel.

## How to use this app:
Each individual *'spark'* message is divided into 6 times of the day;
- morning
- mid-morning
- afternoon
- afternoon snack time
- dinner
- after dinner / night

  In order to get your reccomended *'spark'*, you must input a time of day that is valid to the list above.
